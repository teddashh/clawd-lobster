# Project Scaffold

## Directory Structure

```
<workspace>/
├── CLAUDE.md
├── workspace.json
├── .gitignore
├── knowledge/
├── skills/learned/
├── openspec/
│   ├── project.md
│   ├── changes/
│   └── specs/
├── src/                     # Source code
├── tests/                   # Tests
└── docs/                    # Documentation
```

## Notes

- General-purpose project. No Docker, no deploy pipeline.
- Ship via `git push` only.
- Structure adapts to whatever the project needs.
- This is the simplest workspace type — use when nothing else fits.
