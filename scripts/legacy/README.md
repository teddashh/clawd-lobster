# Legacy Scripts

These are v0 prototypes from the Spec Squad development. They have been superseded by the unified `clawd_lobster` package.

| Script | Replaced By |
|--------|-------------|
| `spec-squad-v1.py` | `clawd-lobster squad start` (terminal) |
| `spec-squad-ui-v1.py` | `clawd-lobster serve` → /squad (web) |
| `spec-squad-viewer-v1.py` | `clawd-lobster serve` → /squad dashboard |
| `spec-squad-sdk-v1.py` | `clawd_lobster/squad.py` (Agent SDK engine) |

These files are kept for reference only. Do not use them for new projects.
